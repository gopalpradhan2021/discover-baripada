# Discover Baripada

Discover Baripada is an Expo + React Native app that curates local routes, food, culture, and nearby places to help you plan a day in the city.

## Quick start

1. Install dependencies

```bash
npm install
```

2. Start the app

```bash
npx expo start
```

## Scripts

- `npm run start` - Start the Expo dev server
- `npm run android` - Open on Android
- `npm run ios` - Open on iOS
- `npm run web` - Open on web
- `npm run lint` - Run lint checks

## Project layout

- `app/` - File-based routes (expo-router)
- `components/` - Reusable UI components
- `constants/` - Theme tokens and shared values
